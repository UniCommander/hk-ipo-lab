# HK-IPO-Lab

> 私人研究系统。目标：一个人就是一支队伍。

## 它是干什么的？
用 AI 辅助我做港股IPO宏观策略研究。只出深度笔记，不做投资决策。

## 文件夹地图
- `framework/` → 研究框架和模板（宏观评分、赛道筛选、个股速评）
- `prompts/` → 给 AI 的指令库（我的专属弹药）
- `research_notes/` → 最终成品笔记（AI信息+我的判断）
- `raw_ai_outputs/` → AI 原始输出（留底备查）

## 核心纪律
- **只做研究员，不做决策者。**
- **只展示思考，不做任何承诺。**
